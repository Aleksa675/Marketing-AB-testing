# A/B тестирование рекламной кампании

## Описание проекта
Анализ результатов A/B теста для оценки эффективности изменения в рекламной кампании. Осуществлялась проверка, что новое рекламное объявление (тестовая группа) приводит к статистически значимому увеличению конверсии по сравнению со старым объявлением (контрольная группа).

## Цель исследования
Определить, следует ли внедрять новое рекламное объявление для всех пользователей на основе статистического анализа результатов A/B теста.

## Данные
- **Источник:** [Kaggle: Marketing AB Testing](https://www.kaggle.com/datasets/faviovaz/marketing-ab-testing)
- **Метрика:** Конверсия (переход по ссылке / покупка)

## Используемые инструменты и библиотеки
- **Python**
- **Pandas** — обработка и анализ данных
- **Matplotlib** — визуализация результатов
- **Statsmodels** — статистические тесты (z-тест)

## Ход работы

### 1. Разведочный анализ (EDA)
- Проверка данных на пропуски и аномалии
- Визуализация распределений
- Сравнение конверсий в группах

### 2. Статистический анализ
- Проведение двухвыборочного z-теста для пропорций
- Расчет p-value
- Интерпретация результатов

**Статистические выводы:**
- Z-статистика: -7.37
- P-value: 1.7e-13

### 3. Визуализация результатов
- Столбчатые диаграммы сравнения конверсий

## Выводы и рекомендации

**Вывод:**  
Разница в конверсии между группами является **статистически значимой** (p-value < 0.05). Тестовая группа показывает конверсию на 43% выше контрольной.

**Рекомендация:**  
Внедрить новое рекламное объявление для всех пользователей.
